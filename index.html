Latitude vs. Weather Factors

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot varius aspects of the weather vs. lattitude. Factors we looked at included temperature, cloudiness, wind speed and humidity. This site provides the source data and visualizations we crated as part of the analysis, as well as explanations and descriptions
